# WebHarvester Chrome Extension Info Page

This folder contains the static public info pages for the **WebHarvester** Chrome extension.

## What this project includes

- `index.html` - Main landing/info page for the extension
- `release-notes.html` - Version history and changes
- `privacypolicy.html` - Privacy policy used for store/public disclosure
- `support.html` - Support and contact page

## Purpose

This site is used as the extension's public documentation/info surface, including:

- Product overview and feature summary
- Release notes for published versions
- Privacy policy details
- Support contact details

## Tech stack

- Plain HTML + CSS (no framework, no build step)
- Static hosting friendly (GitHub Pages or any static host)

## Local preview

Open any page directly in a browser, for example:

- `index.html`
- `release-notes.html`
- `privacypolicy.html`
- `support.html`

Or run a local static server from this folder if preferred.
